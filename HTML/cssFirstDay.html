Ways to Apply CSS

• Inline – CSS is added to the elements directly using the style attribute. • Internal – CSS is within the HTML file, using the style tag. • External – both HTML and CSS are in a separated file. The CSS file should be saved as (.css) extension. After that we are linking the CSS file into html file.
